# Linux Practice - Day 1

## What I Learned
Today I practiced basic Linux commands and understood how they work.

## Commands I Used
- pwd → show current directory
- ls → list files
- cd → navigate between directories
- mkdir → create directory
- touch → create file
- cp → copy files
- mv → move/rename files
- rm → delete files and directories
- cat → view file content

## Important Lesson
I learned that:
- sudo does NOT apply to file redirection (>>)
- To write into protected files, use:
  echo "text" | sudo tee -a filename
